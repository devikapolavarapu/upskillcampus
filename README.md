
This repository contains basic machine learning and data science projects built using real-world datasets.

## Projects

### Traffic Forecasting
- Predicted traffic using time-based features like hour, day, and month  
- Handled categorical data (junctions)  
- Compared Linear Regression and Random Forest models  
- Evaluated using RMSE and MAE  

### Crop Production Prediction
- Predicted crop production using features like state, crop type, season, and area  
- Cleaned data and handled missing values  
- Encoded categorical variables  
- Compared Linear Regression and Random Forest models  
- Evaluated using RMSE and MAE  

## Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn
